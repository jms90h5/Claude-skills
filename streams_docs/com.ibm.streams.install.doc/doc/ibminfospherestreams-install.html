      • Installation packages

        Use the main installation package to install all of the product files on the resources in your Teracloud® Streams domain. To reduce the product size requirement on resources, the domain host installation package contains only a subset of the product files. For example, this installation package does not include the toolkits.

        To run multiple installations of Teracloud® Streams on the same resource, each installation must be installed in a unique location.

      • Automated installation method

        The automated installation method enables you to build an installation package that you can use to install the Teracloud® Streams product on a large number of networked resources that share the same configuration.
