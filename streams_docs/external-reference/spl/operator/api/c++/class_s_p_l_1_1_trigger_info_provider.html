Detailed Description

Interface which provides information about an event-time window, observable within the WindowEvent handler code.

Windows defined over the event time domain implement this interface.

Since
IBM Streams Version 4.3.0

Member Enumeration Documentation

◆ PaneTiming

enum SPL::TriggerInfoProvider::PaneTiming

Enumerates the timing possibilities for the trigger firing related to the current watermark of the enclosing operator.

A window may trigger multiple times, and the timing of those triggers is considered:

  • paneEarly if the window pane is not complete when it triggers.
  • paneOnComplete if the window pane has just completed and, as a result, the pane has triggered.
  • paneLate if the window pane has triggered because late tuples being inserted.
