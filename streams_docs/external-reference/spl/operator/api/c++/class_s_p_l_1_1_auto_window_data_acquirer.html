Detailed Description

template<class T, class G = int32_t, class D = std::deque<T>, class S = std::unordered_map<G, D>>
class SPL::AutoWindowDataAcquirer< T, G, D, S >

Class that provides an RAII handler for window data access.

This class is used for acquiring the data associated with a window, with locking if required. Example usage is as follows:

 { AutoWindowDataAcquirer awda(window);
   ...
 }

This class, when constructed, calls acquireData() on the window, and when destructed, calls releaseData() on the window
