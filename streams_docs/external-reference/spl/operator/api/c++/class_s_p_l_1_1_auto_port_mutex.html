Detailed Description

Class that provides an RAII handler for critical sections in port processing.

Protect a critical section of code using a passed in mutex, only if the operator whose context is provided could be called concurrently on its input ports by the upstream operator or processing element. E.g.:

{ AutoPortMutext apm(myMutex, *this); // Obtain the lock
// Code that manipulates operator state
...
} // Release the lock
