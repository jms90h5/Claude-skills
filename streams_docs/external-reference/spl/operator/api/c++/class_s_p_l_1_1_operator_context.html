◆ registerStateHandler()

virtual void SPL::OperatorContext::registerStateHandler ( StateHandler &  handler )
pure virtual

Registers an object that implements the StateHandler interface to manage the state of the operator.
If the operator is in a consistent region, the SPL Runtime calls the handler's methods to make the operator's state consistent or to reset it to a previous consistent state.
If the operator is checkpointing, the SPL runtime calls the checkpoint and reset methods. If an Operator itself implements the StateHandler, then the SPL runtime automatically registers the StateHandler after the operator's construction.

Parameters
handler object implementing StateHandler
