Detailed Description

An ArithmeticOperation is a pair of an operator and an operand. The operation may be binary or unary. If the operation is binary Applying an ArithmeticOperation to a value will treat the value as the left operand of the operator, and the operand as the right operand. If the operation is unary, the operand is ignored, and the operation is applied to the value.
