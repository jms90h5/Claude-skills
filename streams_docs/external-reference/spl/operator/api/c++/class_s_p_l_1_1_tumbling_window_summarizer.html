Detailed Description

template<class T, class G = int32_t>
class SPL::TumblingWindowSummarizer< T, G >

Class that defines an interface for tumbling window summarizers.

A tumbling window summarizer may be used to remove the need for the Windowing library to retain all tuples in a tumbling window. If your operator uses the windowing library to group and partition tuples, but just needs to look at incoming tuples once, a tumbling window summarizer can be used.

Inherit from this class and override select methods to implement summarizers for tumbling windows.
