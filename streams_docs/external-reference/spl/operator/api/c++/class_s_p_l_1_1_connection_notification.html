Detailed Description

Notification emitted by a ControlPlaneConnector.

A notification is emitted when a client connection is opened, has failed unexpectedly, is closed, or when notifications are lost.

The notification has one of the following types:

Type Meaning
jmx.remote.connection.jcp.setup The Job Control Plane has been setup for this ControlPlaneConnector.
jmx.remote.connection.closed A client connection has been closed.
jmx.remote.connection.failed A client connection has failed unexpectedly.
jmx.remote.connection.notifs.lost A client connection has potentially lost notifications.

A SETUP notification is issued after the connection to the Job Control Plane for the processing element is established or reconnected. In case of reconnection, the ControlPlaneConnector will first re-establish the MBean subscriptions (which were lost when the connection failed), then it will send the SETUP notification. The function handling this notification should return as soon as possible, to avoid blocking its notification emitter.

Since
InfoSphere® Streams Version 4.0.0.
