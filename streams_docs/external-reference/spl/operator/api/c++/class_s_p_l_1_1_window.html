template<class T , class G = int32_t, class D = std::deque<T>, class S = std::unordered_map<G, D>>
virtual StorageType const& SPL::Window< T, G, D, S >::acquireData ( ) const
pure virtual

Get the data associated with this window. If this window involves eviction or trigger policies that are time based, appropriate locking is performed. This call should be followed by a releaseData() call for releasing the lock in these cases. See SPL::AutoWindowDataAcquirer for a safer, RAII based access to the contents of the window.

Returns
contents of the window
