Detailed Description

Classes which provide windowing functionality.

The C++ windowing library provides consistent window policy semantics across operators and simplifies primitive operator implementation when using windows.

The library offers tumbling, sliding, and event-time windows. A window keeps all the incoming data in memory until its tuple eviction policy triggers. The window fires events when significant changes occur in the object's state. Developers can implement the event handling actions independently of the window policy details.
