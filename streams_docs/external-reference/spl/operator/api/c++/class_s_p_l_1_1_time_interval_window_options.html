Detailed Description

Provides configuration options for an event-time window with fixed duration window time intervals.

An event-time window partitions the event-time domain into equally sized intervals of the form:

[N * creationPeriod + intervalOffset, N * creationPeriod + intervalOffset + intervalDuration)

where:

  • intervalDuration defines the duration between the interval start and the end.
  • creationPeriod defines duration between two adjacent intervals, that is, the duration between two consecutive interval start endpoints.
  • intervalOffset defines an interval start endpoint. The value is adjusted by subtracting creationPeriod values until it falls within the interval [0 ... creationPeriod).
  • discardAge defines maximum acceptable tuple lateness. A tuple is late when it is assigned to a window pane which is complete. The discardAge option defines the duration between the point in time when a window pane becomes complete and the point in time when the window does not accept late tuples any longer. Tuples assigned to the window after this point are discarded.

See SPL::TimeIntervalWindow and SPL::TimeIntervalWindowPane for a detailed description of operational semantics of event time windows,
