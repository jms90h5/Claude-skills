◆ assignFrom()

void SPL::Tuple::assignFrom ( Tuple const &  ot,
bool  typesafe = true 
)

Assign this tuple using another tuple.

Parameters
ot Other tuple to assign from.
typesafe If true, expects tuples of the same type, otherwise (false) performs a partial assignment by copying only matching attributes. An attribute matches if the name is the same, and the type is either the same, or the type to be copied from is T and the type to be copied to is optional<T>. Default is false.
Exceptions
SPLRuntimeTypeMismatchException when typesafe assignment is requested and the types do not match
