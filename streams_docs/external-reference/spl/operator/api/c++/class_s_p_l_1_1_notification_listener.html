Invoked when a notification occurs.

The implementation of this method should return as soon as possible, to avoid blocking its notification emitter.

Parameters
notification The notification.
handback Pointer to data which helps the listener to associate information regarding the emitter. This pointer is passed to the addNotificationListener call and resent, without modification, to the listener.
