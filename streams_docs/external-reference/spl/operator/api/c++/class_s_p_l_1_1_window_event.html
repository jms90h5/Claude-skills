Detailed Description

template<class T, class G = int32_t, class D = std::deque<T>, class S = std::unordered_map<G, D>>
class SPL::WindowEvent< T, G, D, S >

Class that defines an interface for window events.

Inherit from this class and override select methods to handle window events you are interested in.

This event is fired right before a TimeIntervalWindowPane transitions from TimeIntervalWindowPane::Complete to TimeIntervalWindowPane::Closed.

The client code cannot insert tuples into closed windows. The handler should use this event for releasing any of its resources associated to this window.

Parameters
window the window on which this event has fired. Can be cast to TimeIntervalWindowPane.
partition the partition that the window trigger applies to (for partitioned windows)

inlinevirtual

This event is fired when partitions are to be selected for removal.

This event is only fired for windows that have a partition eviction policy with a OperatorDefined partition selection. The markForRemoval function should be called by the implementer of this function on the iterators that point to the partitions that are desired to be removed.

Parameters
window the window on which this event has fired
begin iterator to first (oldest) partition
end iterator just past the last (newest) partition
