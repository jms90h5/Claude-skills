Detailed Description

Class that represents a punctuation based window policy.

  • For a tumbling window, as an eviction policy

A window is considered full when a punctuation is received.

Operational semantics: When a tuple is received, it is inserted into the window. When a punctuation is received, the whole window flushes (mass eviction).

Note: Sliding windows cannot have punctuation based eviction or trigger policies.
