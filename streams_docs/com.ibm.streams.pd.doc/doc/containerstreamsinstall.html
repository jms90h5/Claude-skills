          • Installation stops with JRE Libraries error message

            If the Teracloud® Streams installation takes hours to complete rather than minutes, check your system date and time.

          • X11TransSocketINETConnect message during the product installation

            If you install the product by using the console method or the silent method and the program issues an X11TransSocketINETConnect message, you can ignore the message.

            When you try to install Teracloud® Streams from File Browser, you encounter problems and cannot install the product.

            Any application that requires an X Window System cannot be launched.
