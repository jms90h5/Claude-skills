User is not authorized to view the domain configuration

Teracloud® Streams issues the following error message: 
CDISA5095E The user-id user is not authorized to view the domain configuration for the following domain: domain-id.
The certificate user does not have authorization to the domain.

By default, the certificate user is the Subject Common Name. For more information, see Setting the user ID pattern for certificate authentication.

Set up Teracloud® Streams authorization for the certificate user.

You can configure permissions for a certificate user by using roles or by setting access permissions for the user. Groups do not apply to certificate users. To configure permissions, see Configuring user access to Teracloud® Streams domains and instances.
